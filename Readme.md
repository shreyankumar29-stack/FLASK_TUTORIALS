## ▶️ How to Run

Clone the repository:

```bash
git clone <repository-url>
```

Move into the project folder:

```bash
cd Flask_Tutorials
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

**Windows**

```bash
venv\Scripts\activate
```

Install Flask:

```bash
pip install flask
```

Run the application:

```bash
python flaskblog.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
